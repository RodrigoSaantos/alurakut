<h2 id="usage" > ğŸ‘· Usage </h2>

Required! Install git, node and yarn (or npm).

```bash
# Clone Repository
$ git clone https://github.com/RodrigoSaantos/alurakut.git

# Go to server folder
$ cd alurakut

# Install Dependencies
$ yarn
# or
$ npm install

# Run Aplication
$ yarn start
# or
$ npm run start

# Access localhost
http://localhost:3000
```
